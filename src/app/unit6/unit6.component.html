Aufgabe


In dieser Aufgabe sollst du eine eigene DSML entwickeln. Die Konzepte der Domäne wurden bereits identifiziert und werden dir im Folgenden beschrieben. Du sollst diese in ein Metamodell überführen, eine eigene grafische Notation festlegen und anschließend eine Beispielinstanz modellieren. Die Domäne beschreibt eine kleine IT-Umgebung, in der es Computer, installierte Software und angeschlossene Drucker gibt.

Ein Computer besitzt einen eindeutigen Namen. Er kann mehrere Software-Programme installiert haben und mit mehreren Druckern verbunden sein. Jeder Computer muss mindestens eine Software installiert haben und mindestens einen Drucker angeschlossen haben.

Software besitzt ebenfalls einen eindeutigen Namen und ist immer auf mindestens einem Computer installiert. Eine Software kann auf beliebig vielen Computern installiert sein, darf jedoch nicht ohne zugehörigen Computer existieren.

Ein Drucker hat ebenfalls einen eindeutigen Namen und ist immer mit mindestens einem Computer verbunden. Ein Drucker kann mit beliebig vielen Computern verbunden sein, darf jedoch nicht ohne zugehörigen Computer existieren.

Deine Aufgaben:

  1. Erstelle ein passendes Metamodell auf M2 im XModeler, das exakt diese Konzepte und Constraints abbildet.
  2. Entwickle eine grafische Notation für die Modellierung in dieser Domäne.
  3. Modelliere eine beliebige, konkrete Instanz der Domäne mit deiner Notation.
